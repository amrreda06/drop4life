## الجزء التاسع: اختبار API

### اختبار GET (قراءة)
```bash
curl http://localhost:8000/api/sample-analyses/
```
- [ ] الأمر لا يعطي خطأ
- [ ] البيانات تُرجع بصيغة JSON

### اختبار POST (إنشاء)
```bash
curl -X POST http://localhost:8000/api/sample-analyses/ \
  -H "Content-Type: application/json" \
  -d '{"bagId":"TEST","confirmedBloodType":"O+","status":"Pending"}'
```
- [ ] الأمر لا يعطي خطأ
- [ ] Response code = 201

### اختبار PUT (تحديث)
```bash
curl -X PUT http://localhost:8000/api/sample-analyses/BAG-001/ \
  -H "Content-Type: application/json" \
  -d '{"status":"Approved","confirmedBloodType":"O+"}'
```
- [ ] الأمر لا يعطي خطأ
- [ ] Response code = 200
