## الخطوة 2️⃣: إعادة تشغيل الخادم

اضغط في Terminal:
```
Ctrl+C  (لإيقاف الخادم الحالي)
```

ثم اكتب:
```bash
python manage.py runserver
```

**ما تتوقعه:**
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

✅ الخادم يعمل الآن

## الخطوة 3️⃣: فتح المتصفح

أفتح رابط جديد:
```
http://localhost:8000
```

**ما تتوقعه:**
- صفحة تسجيل الدخول

---

## الخطوة 4️⃣: تسجيل الدخول

سجل دخول بـ:
- **Username:** معمل (lab user)
- **Password:** كلمة المرور الخاصة بك

**مثال:**
```
Username: lab_user
Password: password
```

✅ إذا دخلت بنجاح، انتقل للخطوة 5

---

## الخطوة 5️⃣: الذهاب للمعمل

في الـ Dashboard:
```
1. اذهب للقائمة الرئيسية
2. اختر "المعمل" أو "Lab"
3. اختر "الأكياس" أو "Blood Bags"
```

**ما تتوقعه:**
- جدول يظهر به أكياس دم
- بعض الأكياس حالتها "معلق" (Pending)

---

## الخطوة 6️⃣: البحث عن الزر

في الجدول:
```
1. ابحث عن أكياس بحالة "معلق"
2. يجب أن ترى زر "🔬 تحليل"
3. اضغط على الزر
```

**ما تتوقعه:**
- نافذة صغيرة (Modal) تفتح
- فيها اختيارات للفصيلة والأمراض

---

## الخطوة 7️⃣: اختبر القبول

في النافذة:
```
1. اختر فصيلة: "O+"
2. لا تختر أمراض (اختياري)
3. اضغط الزر الأخضر: "✅ قبول العينة"
```

**ما تتوقعه:**
- النافذة تغلق
- رسالة نجاء تظهر
- الجدول يتحدث

---

## الخطوة 8️⃣: اختبر الرفض

في النافذة (لكيس آخر):
```
1. اختر أمراض: ضع علامة "✓" على "HIV"
2. أدخل سبب الرفض: "فيروسات"
3. اضغط الزر الأحمر: "❌ رفض العينة"
```

**ما تتوقعه:**
- النافذة تغلق
- رسالة تظهر (باللون الأزرق)
- الجدول يتحدث

---

## الخطوة 9️⃣: تحقق من الإشعار

سجل دخول كأدمن:
```
1. تسجيل دخول جديد كـ Admin user
2. اذهب للإشعارات
3. ابحث عن الإشعار الجديد
```

**ما تتوقعه:**
- إشعار يظهر عن العينة المرفوضة
- يحتوي على الأمراض والسبب

## 🆘 لو حدث خطأ

### خطأ: "Migration Failed"
```bash
python manage.py showmigrations api
python manage.py migrate api 0010
```

### خطأ: "Port Already in Use"
```bash
# استخدم port مختلف
python manage.py runserver 8001
```

### خطأ: "404 Not Found"
```
تأكد من:
1. الخادم يعمل
2. الرابط صحيح
3. أعد تحميل الصفحة
```

### خطأ: "الزر لا يظهر"
```
Ctrl+Shift+Delete  (امسح الكاش)
Ctrl+Shift+R  (أعد تحميل)
```

## 💡 نصائح مهمة

1. **Migration مرة واحدة فقط**
   - تُطبق مرة واحدة
   - لا تحتاج لتطبيق مجدداً

2. **الخادم يبقى مفتوح**
   - استخدم Terminal آخر للأوامر
   - أو افتح نافذة جديدة

3. **المتصفح قد يخزن الكاش**
   - استخدم Ctrl+Shift+R (Refresh + Clear Cache)
   - أو افتح Incognito Mode

4. **الأمراض والفصائل**
   - 8 فصائل للاختيار من
   - 8 أمراض للاختيار من
   - الاختيار اختياري
